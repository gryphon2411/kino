# Kino

Kino ("Cinema"), derived from Yiddish (קינאָ), represents a personal educational initiative focused on exploring various technologies within the context of a cinema.

## Provision
```bash
$ basename "$(pwd)"
kino
$ cd ./orchestrators/k8s/terraform
$ cp .env.example .env
$ task deploy
```
